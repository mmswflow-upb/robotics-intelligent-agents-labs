# IA Lab 0 — Windows Setup and Study Guide

Important:
Lab 0 is a theory/introduction lab about **Agent-Oriented Programming (AOP)** and **NetLogo**.
The PDF does **not** require a specific operating system.
For Windows, the practical setup you need is a PDF reader and NetLogo.

## What the PDF is about

According to the Lab 0 PDF, the main topics are:
- automation of complex processes
- agent-oriented programming
- intelligent agents
- agent communication
- NetLogo as a multi-agent system modeling environment

The PDF explains that AOP:
- combines ideas from distributed systems, object-oriented programming, and artificial intelligence
- models software as a collection of agents
- lets agents interact to achieve goals
- treats agents as more than simple objects, because they can have beliefs, capabilities, obligations, and the ability to choose

## What you need on Windows

Required:
- a PDF reader
- NetLogo installed

You do **not** need:
- Ubuntu
- ROS
- RViz
- turtlesim
- CoppeliaSim

## Install NetLogo on Windows

1. Open your browser.
2. Download NetLogo from the official NetLogo website.
3. Run the installer, or extract the archive if the download is portable.
4. Install it in the default location, or place it in a simple folder such as `C:\Tools\NetLogo`.
5. Start NetLogo.

Expected:
- NetLogo opens successfully
- you can see the Interface, Info, and Code tabs

## What to study from Lab 0

1. Why agent-oriented programming appeared:
   systems became more numerous, more heterogeneous, and harder to integrate.

2. What an agent is:
   an agent is not just an object; it can communicate, choose actions, pursue goals, and have internal state.

3. The two dimensions mentioned in the PDF:
- agent-oriented programming languages
- agent-oriented communication

4. Advantages of AOP:
- autonomy
- decision-making
- communication of requests, services, intentions, beliefs, and goals

5. Shoham’s three required elements:
- a formal language for agent states and messages
- a programming language with similar semantics
- a way to convert neutral applications into agents

6. NetLogo:
- a multi-agent system modeling environment
- based on the AOP paradigm
- useful later in the course

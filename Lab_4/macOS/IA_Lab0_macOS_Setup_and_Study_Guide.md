# IA Lab 0 — macOS Setup and Study Guide

Important:
Lab 0 is theoretical. On macOS, you mainly need a PDF reader and NetLogo.

## Install NetLogo on macOS

1. Download NetLogo from the official website.
2. Open the `.dmg`.
3. Drag **NetLogo** into **Applications**.
4. Launch NetLogo.
5. If macOS blocks it, go to **System Settings -> Privacy & Security** and choose **Open Anyway**.

Expected:
- NetLogo opens
- you can see Interface, Info, and Code
